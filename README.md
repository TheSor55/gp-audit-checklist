## 📋 เกี่ยวกับเครื่องมือนี้

แบบประเมินแบบโต้ตอบ (Interactive Checklist) สำหรับการตรวจประเมินสถานประกอบการตามเกณฑ์ **Green Production 2569** ครบทุกหมวด พร้อมระบบคำนวณคะแนนแบบ Real-time

### ✅ ฟีเจอร์หลัก

- **GP Core 7 หมวด** — 34 ตัวชี้วัด ครบถ้วนตามเกณฑ์ DCCE 2569
- **Live Scoring** — คะแนนและสถานะอัปเดตทันทีเมื่อกรอกข้อมูล
- **Mandatory Pass Alert** — เตือน 3.2.1 และ 5.2.2 ชัดเจน
- **CE Supplement** — 10 ข้อ ระบบคะแนน 0/1/2 พร้อม Rubric
- **Findings Box** — บันทึกหลักฐานและ Observations ทุกข้อ
- **Print Ready** — พิมพ์ได้ทันทีจากเบราว์เซอร์
- **ไม่ต้องติดตั้ง** — ไฟล์ HTML เดียว ใช้ได้ทุกอุปกรณ์
